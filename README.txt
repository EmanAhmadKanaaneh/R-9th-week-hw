# R Analysis Project

## Description
This project is part of the R introduction course. It includes:
1. A script (`analysis.R`) that creates a dataset and analyzes it.
2. A script (`functions.R`) with a function for descriptive statistics.

## Files
- `analysis.R`: Main script for creating and analyzing data.
- `functions.R`: Contains a function to generate descriptive statistics.
- `README.md`: This file.

## How to Run
1. Load the `analysis.R` script.
2. Ensure the `functions.R` file is in the same directory.
3. Run `analysis.R` to see results and save the output to a file.
